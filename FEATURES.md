# Features

## Design

- Responsive two-column hero layout
- Sticky glass-style header
- Clean card-based sections
- Light and dark theme support

## Interactivity

- Mobile navigation toggle
- Persistent dark mode switch
- Reveal-on-scroll animation for cards
- Contact form validation with inline feedback

## Project qualities

- Built with semantic HTML
- Styled with plain CSS only
- Uses local SVG assets
- Runs without any build step or dependencies
