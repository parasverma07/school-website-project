# Customization Guide

Use this guide to personalize the project quickly.

## Change the school name

- Open `index.html`.
- Update the brand text in the header.
- Update any section headings or contact details that mention the school name.

## Adjust theme colors

- In the `<style>` block, edit the CSS variables inside `:root`.
- For the dark theme, edit the variables inside `body.dark-mode`.

## Replace images

- Add or replace files inside the `assets/` folder.
- Update the `src` attributes in the hero or campus gallery cards.
- SVG files work well for lightweight portfolio projects.

## Edit content sections

- Update the text in the About, Programs, and Campus sections.
- Add or remove feature cards by duplicating existing article blocks.
- Keep headings short for a cleaner layout.

## Update contact information

- Edit the address, email, and phone values in the Contact section.
- Update the success message text if you want a different form response.

## Add more sections

- Insert a new `<section>` block before the closing `</main>` tag.
- Add a matching navigation link in the header.
- Reuse the existing card styles to stay visually consistent.
